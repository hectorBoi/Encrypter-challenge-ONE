Text Encrypter

About the Project

The encrypter follows the next rules using a simple switch statement.

The letter "e" is converted to "enter"
The letter "i" is converted to "imes"
The letter "a" is converted to "ai"
The letter "o" is converted to "ober"
The letter "u" is converted to "ufat"

The project was made using exclusively HMTL, CSS and Javascript as part of the program ONE - "Oracle Next Education"
